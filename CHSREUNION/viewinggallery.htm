Gallery Hints: Viewing the Photo Gallery





  • Go to the Our photo gallery Link.
  • Click on the photo that represents the album you would like to view.
  • Multiple photos will be displayed. You can view them this way, page by page using the next, previous arrows.
  • Or you can click on View Slideshow at the bottom of the box to the left.
  • You will be presented with options across the top of the screen, I recommend you:
    1. Click on Show More Info.
    2. Then use the dropdown arrow to choose the number of seconds you would like each photo to display.
    3. Next use the dropdown arrow to change the max size to 640x640.
    4. If you would like to view a photo for a longer time, click the pause option.
  • At any point in the Gallery, click the large orange Gallery word on top to go back to the initial Gallery page.
  • TO PRINT THIS PAGE, click on File in the upper left corner of this screen, and choose the print option.
